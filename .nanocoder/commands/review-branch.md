---
description: Perform a code review of a specific branch against main branch.
parameters: [branch, main-branch]
---

The {{branch}} branch is part of a PR.  Please perform a code review of the changes against the {{main-branch}} branch.

Use "git diff origin/{{main-branch}}..." to see the changes made. 

Focus on: 

1. Brief summary of the changes
2. Code quality and best practices
3. Potential bugs or edge cases
4. Security considerations
5. Suggestions for improvement
